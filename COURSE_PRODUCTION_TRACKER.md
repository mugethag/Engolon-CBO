Program: Certificate in Robotics, Autonomous Systems and Intelligent Drone Technologies

Purpose: preserve the production workflow already established and use Module 1 and Module 2 as the model for the remaining course.

## Production Standard

Use these completed MVP packages as the model:

- Module 1 package: `robotics-cbi-orientation/course-mvp/term-1/module-1/`
- Module 2 package: `robotics-cbi-orientation/course-mvp/term-1/module-2/`
- Simulation Lab 01: `robotics-cbi-orientation/course-mvp/term-1/lab/simulation-lab-01.md`
- Orientation source: `robotics-cbi-orientation/index.html`
- Narrated orientation render: `robotics-cbi-orientation/renders/program-orientation-narrated.mp4`

Every module should include:

1. `lesson-script.md`
2. `storyboard.md`
3. `quiz.md`
4. `handout.md`
5. HyperFrames lesson composition
6. Rendered MP4
7. Moodle lesson page
8. Moodle quiz
9. Moodle resource downloads

## Status Legend

- `Not Started`: no production file yet
- `Drafted`: written but not reviewed
- `SME Review`: ready for subject-matter review
- `Revision`: changes needed
- `Approved`: content approved
- `In Production`: HyperFrames/video production underway
- `Rendered`: video exported
- `Uploaded`: uploaded to Moodle
- `Live`: available to enrolled students
