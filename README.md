# Engolon Community Based Organization (CBO)

**"Caring for all, leaving none behind."**

A responsive, mobile-first website for Engolon CBO – a Kenya-centered organization serving vulnerable communities: women, children, the elderly, and persons with disabilities (PWDs).

## Overview

Engolon CBO exists to serve and uplift the most vulnerable members of our community through:
- **Women**: Skills training, financial literacy, and entrepreneurship support
- **Children**: Educational support, mentorship, and digital literacy
- **Elderly**: Social support, healthcare access, and government services assistance
- **PWDs**: Accessibility advocacy, assistive devices, and inclusive resources

## Features

✅ **Responsive Design** — Mobile-first approach optimized for all devices  
✅ **Interactive Elements** — Smooth scrolling, mobile menu, form validation  
✅ **Impact Statistics** — Animated counters triggered on scroll  
✅ **Testimonials** — Success stories from community beneficiaries  
✅ **Technology Philosophy** — Explains our balanced approach to AI and digital tools  
✅ **Multiple CTAs** — Donate, Volunteer, Newsletter signup, Contact options  
✅ **Accessibility** — Semantic HTML, ARIA labels, keyboard navigation  
✅ **Performance** — Lightweight CSS, vanilla JavaScript, optimized images

## Getting Started

### View the Website
1. Open `index.html` in a web browser
2. Website is fully functional and responsive

### Local Development
No build process required! The site runs directly in the browser.

**Files to edit:**
- `index.html` — Content and structure
- `assets/css/styles.css` — Styling and responsive breakpoints
- `assets/js/script.js` — Interactive features

### Responsive Breakpoints
- **Mobile**: 320px - 767px (hamburger menu, single column layouts)
- **Tablet**: 768px - 1023px (2-column grids)
- **Desktop**: 1024px+ (4-column grids, full navigation)

## Features in Detail

### Mobile Menu
- Hamburger icon appears on screens < 768px
- Click to toggle menu open/close
- Menu closes when a link is clicked
- Smooth CSS transitions

### Newsletter Form
- Email validation using regex pattern
- Success/error messages displayed
- Clears form after successful submission
- Auto-hides message after 5 seconds

### Impact Counters
- Triggered when section scrolls into view
- Smooth animation from 0 to target number
- Shows formatted numbers (e.g., "500+")
- Uses Intersection Observer API for performance

### CTA Buttons
- Donate, Volunteer, Contact buttons show placeholder actions
- Resources button prepared for file downloads
- Can be integrated with backend services
