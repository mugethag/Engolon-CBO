# Module 2 Quiz

Module: Introduction to Drone Systems, Aerial Robotics and Control Design Fundamentals

Recommended passing score: 70%

## Questions

1. A drone is best understood as:
   - A. Only a flying camera
   - B. An aerial robotic system with sensing, control, propulsion, communication, and payload subsystems
   - C. A computer that does not move
   - D. A ground vehicle with wheels

2. Which component is primarily responsible for processing sensor data and sending commands to the motors?
   - A. Flight controller
   - B. Frame color
   - C. Landing pad
   - D. Carrying case

3. Roll describes:
   - A. Rotation around the drone's vertical axis
   - B. Tilting left or right
   - C. Moving straight upward only
   - D. Battery charging speed

4. Yaw describes:
   - A. Rotation around the drone's vertical axis
   - B. Tilting forward or backward
   - C. The weight of the payload
   - D. The image resolution of a camera

5. In feedback control, the controller compares:
   - A. Desired behavior with actual behavior
   - B. Logo color with battery color
   - C. Course schedule with student grades
   - D. The drone frame with the packaging

6. The proportional term in a PID controller responds mainly to:
   - A. Present error
   - B. Accumulated historical error only
   - C. The drone's brand name
   - D. Weather reports only

7. State-space control uses state variables such as:
   - A. Position, velocity, orientation, and angular velocity
   - B. Price, logo, packaging, and marketing copy
   - C. Student ID, course name, and password
   - D. File size and font size

8. A basic autonomous navigation stack may include:
   - A. Perception, localization, planning, control, and mission monitoring
   - B. Printing, filing, copying, scanning, and stapling
   - C. Painting, packaging, selling, invoicing, and shipping
   - D. Guessing, drifting, ignoring, and stopping

9. If a drone overshoots its target altitude and oscillates, the issue is most closely related to:
   - A. Stability and controller tuning
   - B. The spelling of the mission name
   - C. The color of the propellers
   - D. The file format of the syllabus

10. Which mission would likely prioritize high-resolution imaging and stable flight near structures?
   - A. Bridge inspection
   - B. Random indoor decoration
   - C. Offline word processing
   - D. Classroom seating arrangement
