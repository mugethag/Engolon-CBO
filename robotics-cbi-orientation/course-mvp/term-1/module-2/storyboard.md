# Module 2 Storyboard

Module: Introduction to Drone Systems, Aerial Robotics and Control Design Fundamentals

Target video duration: 7-8 minutes

| Scene | Time | Purpose | Visual Direction |
| --- | ---: | --- | --- |
| 1. Title | 0:00-0:30 | Introduce drone systems and control design. | Drone silhouette with subsystem callouts. |
| 2. Learning outcomes | 0:30-1:05 | State measurable outcomes. | Five outcome tiles. |
| 3. Drone architecture | 1:05-2:00 | Explain frame, propulsion, power, sensors, controller, communication, payload. | Labeled drone architecture diagram. |
| 4. Flight control | 2:00-2:55 | Explain roll, pitch, yaw, altitude. | Four-axis motion diagram. |
| 5. Control design concepts | 2:55-3:45 | Connect model, command, response, stability, error. | Command-response graph. |
| 6. PID and state-space | 3:45-4:45 | Introduce PID and state variables. | PID block and state vector display. |
| 7. Autonomous navigation | 4:45-5:45 | Explain perception, localization, planning, control. | Navigation stack pipeline. |
| 8. Real-world applications | 5:45-6:45 | Compare missions and design priorities. | Application matrix: environment, infrastructure, disaster, defense. |
| 9. Checkpoint | 6:45-7:15 | Ask altitude oscillation question. | Altitude line graph with overshoot. |
| 10. Recap | 7:15-7:45 | Reinforce takeaways and bridge to lab. | Drone architecture plus feedback loop. |

## Reusable Scene Notes

- Keep flight dynamics conceptual in this MVP.
- Avoid operational guidance for harmful drone use.
- Defense examples should remain analytical and ethical, not tactical.
- Use the checkpoint to prepare learners for the simulation lab.
