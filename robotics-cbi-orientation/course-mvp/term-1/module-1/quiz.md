# Module 1 Quiz

Module: Introduction to Sustainability Robotics and Intelligent Systems

Recommended passing score: 70%

## Questions

1. Which set best describes the core subsystems of a basic robotic system?
   - A. Battery, frame, paint, and packaging
   - B. Sensors, actuators, controller, and feedback
   - C. Screen, keyboard, database, and printer
   - D. Camera, antenna, propeller, and landing gear

2. What is the main role of a sensor in a robotic system?
   - A. To create motion
   - B. To collect information from the robot or environment
   - C. To replace the controller
   - D. To store the final report

3. Kinematics primarily focuses on:
   - A. The motion of objects without emphasizing the forces that cause the motion
   - B. The legal requirements for drone operations
   - C. The cost of robot manufacturing
   - D. The encryption of sensor data

4. Dynamics helps engineers understand:
   - A. Only the color and shape of a robot
   - B. Forces, mass, acceleration, torque, and energy
   - C. Only the user interface of a robot
   - D. Only wireless communication range

5. What is the main difference between open-loop and closed-loop control?
   - A. Closed-loop control uses feedback to adjust behavior
   - B. Open-loop control is always more accurate
   - C. Closed-loop control does not use sensors
   - D. Open-loop control requires artificial intelligence

6. Why is simulation useful in robotics learning?
   - A. It removes the need to understand robotic behavior
   - B. It allows safe, low-cost testing before physical deployment
   - C. It guarantees perfect real-world performance
   - D. It only applies to entertainment robots

7. Which example best fits sustainability robotics?
   - A. A robot designed only to display advertisements
   - B. A drone fleet used to monitor biodiversity and ecosystem change
   - C. A machine built with no concern for energy use or maintenance
   - D. A robot that cannot be repaired or reused

8. Bio-inspired robotics means:
   - A. Copying animals exactly in every detail
   - B. Using ideas from living systems to inform robotic design
   - C. Avoiding natural systems in engineering
   - D. Using robots only in biology laboratories

9. If a mobile robot drifts away from its intended path, which concept is most directly involved in correcting the error?
   - A. Feedback
   - B. Packaging
   - C. Color selection
   - D. Course enrollment

10. A sustainable robotic design should consider:
   - A. Energy use, maintenance, repairability, context, and long-term impact
   - B. Only speed
   - C. Only physical appearance
   - D. Only whether the robot uses a camera
