# Module 1 Storyboard

Module: Introduction to Sustainability Robotics and Intelligent Systems

Target video duration: 7-8 minutes

| Scene | Time | Purpose | Visual Direction |
| --- | ---: | --- | --- |
| 1. Title | 0:00-0:30 | Introduce module and relevance. | Title frame with robot subsystem diagram: sense, decide, act, feedback. |
| 2. Learning outcomes | 0:30-1:05 | State measurable outcomes. | Five outcome tiles. |
| 3. Robotic system basics | 1:05-1:55 | Explain sensors, actuators, controller, feedback. | Four-part loop diagram with animated arrows. |
| 4. Kinematics and dynamics | 1:55-2:50 | Distinguish motion description from force explanation. | Split-screen: path trace versus force/torque labels. |
| 5. Control and simulation | 2:50-3:45 | Explain open-loop and closed-loop control. | Before/after control response line. |
| 6. Sustainability robotics | 3:45-4:50 | Connect robotics to environmental and social applications. | Environmental monitoring, infrastructure, agriculture, disaster icons. |
| 7. Bio-inspired systems | 4:50-5:40 | Introduce nature-inspired design. | Bird, insect swarm, fish, walking gait abstraction. |
| 8. Applied example | 5:40-6:45 | Apply concepts to water-channel inspection robot. | Scenario diagram with terrain, water channel, robot, sensor zones. |
| 9. Checkpoint | 6:45-7:15 | Ask diagnostic reasoning question. | Four subsystem cards. |
| 10. Recap | 7:15-7:45 | Reinforce main takeaways. | Summary loop: sense, act, control, feedback, sustainability. |

## Reusable Scene Notes

- Use the same title and outcomes scene style as the orientation video.
- Keep all diagrams simple enough to render in HTML/CSS.
- Avoid hardware-specific claims unless a later SME review confirms them.
- The checkpoint can become an LMS discussion question or embedded quiz prompt.
