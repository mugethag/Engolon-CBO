Week 1 simulator activity

UAIbot Starter Lab

Your first hands-on session with the UAIbot simulator. Follow the steps below, complete each task, and use the checklist to confirm your submission is ready.

Open UAIbot simulator Back to portal

Step-by-step instructions

Complete all six steps before submitting.

  1. Open the simulator and create your workspace

    Go to uairobot.com and sign in (or continue as guest). Create a new project and name it Cohort1-YourFirstName. Take a screenshot of the empty workspace showing your project name.

    Screenshot required: empty named workspace

  2. Place a UAIbot on the grid

    From the component panel, drag a UAIbot onto the simulation grid. Position it at the centre of the arena. Note the starting coordinates shown in the properties panel — you'll need these in Step 5.

    Remember: X and Y start coordinates

  3. Program a forward movement sequence

    Using the block editor or code view, write a sequence that moves the robot forward 3 steps, turns right 90°, then moves forward 3 more steps. This traces an L-shape path. Run the simulation and observe the robot's movement.

    Tip: use move(steps) and turn("right") commands

  4. Add a loop to repeat the L-shape

    Wrap your movement sequence in a loop that repeats it 4 times. The robot should trace a square-ish path back to its starting position. Run the simulation again and confirm it returns to the origin.

    Screenshot required: completed loop path on grid

  5. Add an obstacle and observe the response

    Place one obstacle block anywhere along the robot's path. Run the simulation and observe what happens when the robot encounters it. Note whether the robot stops, reports an error, or ignores it. Write one sentence describing what you observed.

    Written observation required (1 sentence minimum)

  6. Export your code and save your project

    Export or copy your code from the editor into a plain text file. Save the project in UAIbot. You will attach both the screenshot(s) and the code snippet in your submission email.

    Deliverables: 2 screenshots + code snippet
