Week 1 simulator activity

BrowserBotics Starter Lab

Your first hands-on session with the BrowserBotics simulator. Write Python code to control a virtual robot — no installation needed. Follow the steps below, complete each task, and use the checklist to confirm your submission is ready.

Open BrowserBotics sandbox Back to portal

Step-by-step instructions

Complete all six steps before submitting.

  1. Open the sandbox and run your first program

    Go to browserbotics.com/sandbox. The page loads a Python editor on the left and a 3D simulator on the right. Paste the code below into the editor and click the Run (▶) button: 

    import browserbotics as bb
bb.addGroundPlane()
bb.createBody('box', halfExtent=[0.1,0.1,0.1], position=[0,0,0.1], color='red')
    Screenshot required: simulator window showing the red box

  2. Load a robot and read its starting position

    Replace your code with the program below and run it. The robot's x, y, z position will print in the output panel beneath the editor. Write down those three numbers — you will need them in Step 5. 

    import browserbotics as bb
bb.addGroundPlane()
robot = bb.loadURDF('roomba.urdf')
pos, rot = bb.getBasePose(robot)
print('Starting position:', pos)
    Remember: starting x, y, z values from the output panel

  3. Drive the robot in an L-shape

    Extend your code to drive the robot forward, spin it right 90°, then drive forward again. Use velocity control: equal wheel speeds move forward; opposite speeds spin in place. Add import time and use time.sleep(seconds) between commands so the simulation has time to move. Run it and watch the robot trace an L.

    Tip: bb.setJointMotorControl(robot, 0, controlMode='velocity', targetVelocity=3) — joint 0 is the left wheel, joint 1 is the right wheel

  4. Use a loop to repeat the L-shape 4 times

    Wrap your drive-and-turn code inside a for loop that repeats 4 times. The robot should trace a roughly square path and return close to where it started. Run it and observe the full path.

    Screenshot required: simulation showing the completed 4-loop path

  5. Add an obstacle and observe the response

    Place a stationary box in the robot's path by adding this line before your loop, then run the simulation. Observe what happens when the robot reaches the box — does it stop, push it, or drive through? Write one sentence describing what you saw. 

    bb.createBody('box', halfExtent=[0.15,0.15,0.15],
              position=[0.5,0,0.15], mass=0, color='orange')
    Written observation required (1 sentence minimum)

  6. Save your code as a snippet

    In the sandbox toolbar, click Save Snippet and name it Cohort1-YourFirstName. This creates a shareable link to your code. Copy the snippet URL — you will paste it into your submission email alongside your screenshots and written observation.

    Deliverables: snippet URL + 2 screenshots + written observation
