Week 4 final project

Design a robot that solves a local problem.

Learners may work individually or in small teams. The project should show a clear problem, a robotic behavior, and an explanation of how the robot senses, decides, acts, and uses feedback.

Project options

  • Farm-assist robot
  • Delivery robot
  • Warehouse robot
  • Security patrol robot
  • Hospital supply robot
  • Waste-sorting robot

Submission package

  • Project title and local problem
  • Screenshot or short video from simulation
  • Sensors, actuators, controller, and feedback
  • Short explanation of robot logic
  • Safety or sustainability benefit

Technical explanation

Can the learner explain how the robot senses, decides, acts, and corrects behavior?

Simulation evidence

Does the project include visible evidence of a robot behavior or control idea?

Local relevance

Is the problem connected to agriculture, logistics, health, safety, environment, or community needs?

Level 2 readiness

Does the learner show motivation, clarity, and enough technical confidence for the next level?
